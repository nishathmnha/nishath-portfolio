# Nishath Portfolio

Lightweight React + Vite portfolio site built for simple static deployment.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal.

## Build for deployment

```bash
npm run build
```

The production-ready static files will be generated in `dist/`.

## Free deployment options

- Vercel
- Netlify
- Cloudflare Pages
- GitHub Pages

## Updating content

- Edit default portfolio content in `src/content/portfolioData.js`
- Replace profile image in `public/profile.jpg`
- Replace CV in `public/Nishath_Ahamed_CV.pdf`

## Adding content

- Update `currentWork` for what you are currently working on
- Add showcase videos in `demoVideos`
- Add project links in each project's `links` array
- Add certificate links in `certifications`
